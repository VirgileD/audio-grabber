# Resources Folder

This folder will contain the application icons in Phase 3.

## Required Icons

1. **icon_idle.ico** - Gray microphone icon (idle state)
2. **icon_recording.ico** - Red microphone icon (recording state)
3. **icon_error.ico** - Microphone icon with warning symbol (error state)

All icons should be 16x16 and 32x32 pixels for proper system tray display.

For now, the application uses system icons as placeholders.
